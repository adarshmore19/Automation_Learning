# Automation_Learning
Automation Learning Roadmap

DAY 1-Learnt about 
                   
                   > os.listdir -> Which returns list of all files and folders from the given directory
                   
                   > enumerate -> Which iterates and gives both index and value
                   
                   > os.path.join -> which safly combines path parts into valid full file path
                   
                   > os.rename -> which can rename or move the file from old location to new location

DAY 2-Learnt about 
                   
                   > taking user input and storing that input into variable to change the file name into the prefix given by the user

DAY 3-Learnt about 
                   
                   > os.path.splitext ->Splits the path into pair - root and ext where root is path of file and ext is extension of file

DAY 4- Learnt about 
                   
                   > os.makedirs -> is a Python command used to create folders.
                   > os.path.isdir -> is a Python command used to check if something is a folder
